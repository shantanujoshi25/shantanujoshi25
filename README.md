## Featured Projects

### [RecruiTech](https://github.com/shantanujoshi25/RecruiTech) — AI-Powered Technical Recruitment Platform
> **Python · FastAPI · React · Apache Kafka · CrewAI · GraphQL · PostgreSQL**

Built the end-to-end AI/ML pipeline for an agentic recruitment platform that automates candidate evaluation. Engineered a Kafka-based event streaming architecture, CrewAI multi-agent system for resume and GitHub analysis, GraphQL APIs with Strawberry/FastAPI, and a dark-themed React frontend with role-based access control.

### [RGB-Thermal Fusion YOLOv11](https://github.com/shantanujoshi25/rgb-thermal-fusion-yolov11) — Computer Vision Research
> **Python · PyTorch · YOLOv11 · GANs**

Implemented a GAN-based image fusion pipeline (U-Net generator + PatchGAN discriminator) for RGB-thermal object detection on the M3FD dataset. Achieved **72.7% mAP@50** with YOLOv11n on fused outputs, outperforming single-modality baselines.

### [WebScout](https://github.com/shantanujoshi25/webscout) — Web Security Scanner
> **Python · Security · NSD Certified**

Automated web vulnerability scanning tool. Earned an NSD Certification in Penetration Testing from ISAC for this work.

### [StackComply](https://github.com/shantanujoshi25/StackAuthHackathon) — AI-Powered Compliance Scanner · 🏆 Stack Auth Hackathon
> **Next.js · TypeScript · OpenAI · Stack Auth · Tailwind CSS**

AI compliance scanner that analyzes GitHub repos for data privacy violations. Three AI agents run in sequence — Schema Detective parses database schemas (Prisma, Drizzle, TypeORM, raw SQL), Data Classifier identifies sensitive fields via regex + GPT validation, and Regulation Mapper checks against GDPR, HIPAA, CCPA, PCI-DSS, and SOC2. Outputs a compliance grade (A–F) with risk estimates and actionable recommendations.

### [Bingo with Aliens](https://github.com/shantanujoshi25/Bingo) — Real-Time Multiplayer Game · 🏆 Top 5 at Alien.org Hackathon
> **React · TypeScript · FastAPI · Redis · WebSockets**

Built a real-time multiplayer Bingo mini app during the Alien.org Hackathon (ESI). Implemented authentication, deployment pipeline, and live game state synchronization. [Live Demo →](https://bingo-zeta-eight.vercel.app)

### [Lyft BayWheels Discord Bot](https://github.com/shantanujoshi25/Lyft-BayWheels-Discord-Bot) — Real-Time Transit Monitoring

Discord bot for real-time monitoring of Bay Wheels bike-share station availability around the SJSU campus.
